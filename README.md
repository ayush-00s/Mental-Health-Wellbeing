### Installation

1. **Set up the Frontend (React + TailwindCSS):**
    -cd Wellnest
    -npm install
    -npm run dev

2. **Set up the Backend (Express.js):**
   -cd backend
   -npm install
   -npm run start

3. **Set up the Mental Model Server (Python + Flask):**
   -cd mental-model
   -pip install -r requirements.txt
